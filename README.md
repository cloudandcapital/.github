# Cloud & Capital

**Decision systems for finance and engineering leaders.**

Cloud, SaaS, and AI infrastructure — the full picture. We build tools that make cost visible before the invoice arrives, and frameworks that turn visibility into decisions.

### The Six-Tool Pipeline

`Visibility → Variance → Tradeoffs`

A finance-native FinOps toolkit covering the full scope of the FinOps Foundation 2026 framework.

| Tool | Scope |
|------|-------|
| [finops-lite](https://github.com/cloudandcapital/finops-lite) | Multi-cloud cost visibility · FOCUS 1.0 export |
| [finops-watchdog](https://github.com/cloudandcapital/finops-watchdog) | Anomaly detection across any cost CSV |
| [recovery-economics](https://github.com/cloudandcapital/recovery-economics) | Resilience cost modeling and scenario compare |
| [ai-cost-lens](https://github.com/cloudandcapital/ai-cost-lens) | OpenAI / Anthropic / Bedrock spend · FOCUS 1.0 |
| [saas-cost-analyzer](https://github.com/cloudandcapital/saas-cost-analyzer) | License utilization · per-seat cost · forecasting |
| [tech-spend-command-center](https://github.com/cloudandcapital/tech-spend-command-center) | Unified executive report across all scopes |

All repos: CI on every push. Changelogs, contributing guides, and worked examples.
